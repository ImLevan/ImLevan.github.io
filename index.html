MIS PROYECTOS

Título: KarU - Compra y Venta

Página Web


Descripción:

El proyecto KarU es una página web de compra y venta de vehículos, diseñada para satisfacer las necesidades de los usuarios que buscan una forma fácil, rápida y segura de adquirir o vender un automóvil. El proyecto se realizó en el marco de una asignatura académica de la universidad, utilizando la metodología SAFE en combinación con Scrum, una forma ágil y flexible de gestionar el desarrollo de software. El proyecto se dividió en cuatro módulos, nuestro equipo se encargó del módulo administrativo.

Título: Página Web de Maratón: Salud y Deporte en Sincronía

Diseño web inscripcion a maraton


Descripción:

Un apasionante proyecto universitario desarrollado por tres compañeros que ofrece una página web de maratón única. Con HTML, CSS y JavaScript, creamos una experiencia interactiva con información detallada del evento, mapas interactivos de centros de salud donde los participantes podian ver donde hacerce los chequeos medicos, visualización del circuito donde se pueden observar en los tiempos que llegaron a las ubicaciones clave y la posibilidad de registrar y compartir los tiempos de los corredores.

Título: Aplicación del Clima - WeatherUnlocked API

Diseño Web API


Descripción:

Mí pagina web del Clima es un proyecto personal desarrollado íntegramente por mí, abarcando diseño, desarrollo y gestión. Utilizando HTML, CSS y JavaScript puro, he creado una página que consume la API de WeatherUnlocked para brindar información meteorológica precisa y actualizada de ciudades de todo el mundo. La interfaz intuitiva, detalles completos del clima y la posibilidad de personalizar las ciudades favoritas hacen que esta aplicación sea una herramienta valiosa para estar preparado ante cualquier condición climática.

Título: Buscador de Películas

Comunicación co API de peliculas


Descripción:

Bienvenido a mi proyecto personal, un Buscador de Películas que te sumergirá en el fascinante mundo del cine. Con HTML, CSS y JavaScript, he desarrollado una página web interactiva que te permite buscar películas y acceder a una amplia variedad de información sobre cada una de ellas. Título: Adivina la cancion

Juego en Python

Descripción:

Bienvenido a mi proyecto académico desarrollado en Python con Pygame. Sumérgete en el mundo de la música mientras te diviertes adivinando el nombre de las canciones o sus autores. El juego te desafía a reconocer las letras de diversas canciones, otorgándote puntos por cada acierto. Con un sistema de puntos para aciertos y fallos, sentirás la emoción de competir contra el tiempo o alcanzar un límite de canciones. Al finalizar, se mostrará tu puntaje, permitiéndote comparar tus conocimientos musicales y ofreciéndote la opción de volver a jugar y mejorar tus habilidades.

Título: Naruto Delivery Challenge

Juego en JAVA


Descripción:

Mi proyecto personal académico desarrollado en Java! Basado en el mundo de Naruto sobre un personaje, encargado de realizar entregas en distintos puntos del mapa. Título: Ahorcado

Juego en JAVA


Descripción:

¡Bienvenido al proyecto académico Ahorcado, un juego de ahorcado desarrollado por un equipo de tres personas, incluyéndome! Juego creado con Java y la interfaz Swing. En Ahorcado, te enfrentarás a palabras ocultas representadas por espacios en blanco que deberás completar adivinando letras. Con un sistema de vidas, deberás tener cuidado y tomar decisiones estratégicas para evitar quedarte sin intentos. El juego ofrece la opcion de jugar en ingles o español segun tu preferencia. Pon a prueba tus habilidades y suma puntos a medida que resuelves palabras con precisión.
